# Dataset Placement

Download the dataset referenced in the assignment and place it locally under this directory.

Simple convention:

- `data/raw/`: frame sequences
- `data/labels/`: ground-truth annotations for first-frame bounding boxes

Keep the raw dataset out of version control. This repository already ignores the contents of those folders.
